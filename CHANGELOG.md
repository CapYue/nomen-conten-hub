### 功能模块

#### 🏠 内容库（首页）
- 选题卡片展示（标题、状态、正文预览、标签）
- **双模式发布切换**：
  - 🤖 自动发布：直接复制文案，1秒完成
  - 👤 人工审核：进入完整审核流程
- 状态标签：规划中 / 已生成 / 审核 / 已发布
- 一键复制正文 / 复制封面提示词

#### 🔍 人工审核发布页（全屏）
- **图片选择区**：4张图同时展示，可逐张刷新
  - 点击「🔄 刷新」重新生成该张封面图
  - 点击图片选中（绿色边框+✓）
  - 支持「全部刷新」批量重新生成
- **文案编辑区**：
  - 标题可编辑
  - 正文 textarea 可直接修改
  - 复制按钮同步最新内容
- **发布按钮**：
  - 自动复制完整文案到剪贴板
  - 标记选题为「已发布」
  - 提示打开小红书 App 粘贴+配图

#### 📅 内容日历
- 按月份（4月/5月/6月）分组展示选题
- 可直接在日历上修改每篇计划的发布日期

#### ✨ AI 生成页
- 输入选题方向 → Deepseek API 生成完整内容
- 自动解析返回 JSON（标题+正文+封面描述）
- 一键「加入内容库」创建新选题
- 5个预置爆款选题快速生成

#### ⚙️ 配置页
- Deepseek API Key + API 地址配置（支持自定义代理）
- 飞书 Chat ID / Webhook 配置（审核通知）
- 完整工作流说明

## 🏗️ 技术架构

| 技术 | 说明 |
|------|------|
| 前端 | React + TypeScript + Vite |
| 状态管理 | React useState（本地存储 localStorage 持久化） |
| 样式 | 内联 style 对象（无外部依赖） |
| AI 文案 | Deepseek API（`deepseek-chat` 模型） |
| AI 封面图 | 预生成 CDN 图（每次发布前批量生成） |
| 部署 | Vite build → minimaxi CDN |
| 内容管理 | Git 版本控制，可回溯任意版本 |
